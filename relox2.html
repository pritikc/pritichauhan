Project Overview

Relox Virtual Lab (Mobile App Design)

In collaboration with Relox Virtual Labs, I embarked on a transformative journey to revolutionize healthcare diagnostics through intuitive user experience design.

The project aimed to enhance the accessibility, accuracy, and efficiency of pathology reporting, ultimately speeding up the diagnosis and treatment process for patients worldwide.

Problem Statement

The Relox Virtual Lab's mobile app and admin portal had a poor user experience, with users reporting difficulties in navigation and lack of intuitive controls. The goal was to redesign both platforms to enhance usability and engagement.

Objectives

  • Improve navigation and user experience on the mobile app and admin portal.
  • Increase user engagement and reduce bounce rates.
  • Ensure consistency and usability across mobile and desktop interfaces.

Research and Insights

User Research
  • Improve navigation and user experience on the mobile app and admin portal.
  • Increase user engagement and reduce bounce rates.
  • Ensure consistency and usability across mobile and desktop interfaces.
Key Findings
  • Users found the app navigation complex and unintuitive.
  • Admins struggled with the lack of clear controls and streamlined workflows.

Design Process

Wireframes and Sketches
  • Developed wireframes focusing on improving navigation and usability.
  • Created sketches for a more intuitive admin portal layout.
Prototypes
  • Created interactive prototypes using Figma.
  • Included CSS3 animations and transitions for a more engaging user experience.
Iterative Process
  • Conducted multiple rounds of usability testing with users and administrators.
  • Refined the designs based on feedback, focusing on improving clarity and ease of use.

Visual Design

High-Fidelity Mockups:
  • Designed high-fidelity mockups with a modern, cohesive look.
  • Ensured consistency in typography, color palette, and UI components.
Design System:
  • Developed a design system with standardized UI components, ensuring consistency across the mobile app and admin portal.
  • Elements included buttons, forms, icons, and navigation elements.

Usability Testing and Feedback

Testing Methods:
  • Conducted usability tests with 10 mobile app users and 5 admin portal users.
  • Used task scenarios such as navigating the app, managing user accounts, and accessing reports.
Feedback Incorporation:
  • Improved the app navigation based on user feedback, adding clearer icons and labels.
  • Streamlined admin portal workflows based on feedback, enhancing the user interface and experience.
Iterative Process
  • Conducted multiple rounds of usability testing with users and administrators.
  • Refined the designs based on feedback, focusing on improving clarity and ease of use.

Results and Impact

Metrics and Outcomes:
  • 30% increase in user engagement.
  • 10% decrease in bounce rates.
Learnings
  • Highlighted the importance of iterative design and incorporating user feedback.
  • Recognized the value of animations and transitions in enhancing user experience.
