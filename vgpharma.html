VG Pharma

Mobile App Design

A mobile design that seamlessly integrates holistic care for diabetes, thyroid, and endocrine disorders, offering personalized treatment solutions.

The app also features in-house exercise programs, specialized services like foot care, and a diabetic cafeteria, enhancing the user experience for comprehensive health management.

Project Summary

I led the end-to-end design process to create a clean and intuitive interface that solves key user pain points. This resulted in a 20% increase in trade activity, 40% increase in user retention, 95% user satisfaction rate and no incidence of security breaches.

Design Process
Phase 1
Empathize

Understand the user problem through User Interviews and Research Expert opinions

Phase 2
Specify

Analyze Findings, identify goals and prioritize features based on findings

Phase 3
Ideate

Create low fidelity wireframes to explore design posibilities

Phase 4
Test

Test the design against users to gather feedback and ensure it solves a problem

Phase 5
Design System

Creating the Tone, Color Palette, Iconography and Typography of the brand

Phase 6
Finalize

Based on feedback from users, generate final designs for the mobile and desktop app
